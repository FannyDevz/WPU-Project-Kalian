     **fityannugroho**

    Halo Pak Dhika dan rekan-rekan semuanya! Perkenalkan, saya Fityan, full-stack software engineer.

    Saya ingin membagikan project open API yang saya kembangkan, yaitu:
    idn-area, API yang menyediakan informasi wilayah administrasi di Indonesia mulai dari tingkat provinsi, kabupaten/kota, kecamatan, hingga desa/kelurahan. 

    Link project: https://github.com/fityannugroho/idn-area
    Link demo: https://idn-area.up.railway.app/

    Fitur-Fitur
    Menggunakan data resmi terbaru yang dikeluarkan oleh pemerintah.
    API endpoint yang terdokumentasi dengan baik dan mudah digunakan untuk mengambil data provinsi, kabupaten/kota, kecamatan, dan desa/kelurahan.
    Menggunakan rangkaian tes untuk memastikan validitas sumber data dan juga data keluaran dari setiap endpoint.
    API playground untuk mencoba endpoint dan memeriksa respon (lihat /docs endpoint).

    API ini dikembangkan menggunakan framework NestJS dengan bahasa pemrograman TypeScript, ORM Prisma dan database MongoDB. Kedepannya berencana untuk mendukung jenis database lain seperti MySQL, PostgreSQL, dll.

    Jika rekan-rekan menemukan bug, atau ingin ada fitur baru, atau peningkatan-peningkatan lainnya, jangan sungkan untuk berkontribusi ya.
